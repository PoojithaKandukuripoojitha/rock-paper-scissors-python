# Rock Paper Scissors Game (Python)

## Project Description
This is a simple Rock Paper Scissors game built using Python. 
The user plays against the computer and the program randomly selects rock, paper, or scissors.

## Technologies Used
- Python
- Random Module

## Features
- User vs Computer game
- Random choice generation
- Displays winner based on game rules

## How to Run
1. Install Python
2. Run the file using:
python rock_paper_scissors.py
